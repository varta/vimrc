NERDTreeGit
===========

This is a fork of [NERDTree](https://github.com/scrooloose/nerdtree) that
supports Git.

Dependencies
------------

- [Fugitive](https://github.com/tpope/vim-fugitive)

Features
--------

- Defaults to opening new files in tabs
- If in Git repo, displays the branch.
